Good communication is absolutely essential for a successful project, so when we kickoff a new project we make sure our clients are always in the loop.

This is how we recently advised a client:

For an instant messaging system our team members use a native client that we keep open all the time. This is easier than trying to keep it around in a browser tab. Some of us use http://propaneapp.com, others use http://giantcomet.com/flint. There are also other clients for other platforms and other add-ons at http://campfirenow.com/extras.

Something we also like to do on a project is hook Campfire up to Github: then the Campfire room gets a notification when commits are pushed, pull requests opened/closed etc. It makes it easier for everyone to follow updates to the codebase.
